◆ removeSpaces()

void Cell::removeSpaces ( )

Member function used in the CommandLine.

When a table is read from a file, the unneccessary spaces should be removed. Nonetheless, when the cell value is string or formula, spaces inside it should't be removed. This may cause outputs such as either string, interpreted differently than expected, or wrong formulas.
